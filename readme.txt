== Description ==
This plugin allows you to disable right click and word selection on posts or pages. MHR Anti Copy comes with flexible custom settings. You can protect a single page/post or overall pages. Everything can be done with a few simple steps. With shotcodes you can protect a single page or post easily.

== Installation ==

The automatic plugin installer should work for most people. Manual installation is easy and takes fewer than five minutes.

1. Upload the entire folder of MHR-Custom-Anti-Copy to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to **Settings** and then **MHR-Anti-Copy-CUstom** to configure and enable.
